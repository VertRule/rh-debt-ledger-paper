# R28: Zenodo DOI Record

## Purpose

This rung records the Zenodo DOIs minted via GitHub integration, enabling formal citation of the repository as a citable preprint.

## DOIs

| Type | DOI | Resolves To |
|------|-----|-------------|
| Version | `10.5281/zenodo.18318664` | r28-zenodo-doi release |
| Concept | `10.5281/zenodo.18318663` | Always latest version |

## Non-Claims

- This DOI does **not** indicate peer review
- This DOI does **not** indicate arXiv acceptance
- This DOI provides a citable, permanent identifier only
